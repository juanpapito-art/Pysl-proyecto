PySI - Prototipo Funcional de Sistema de Información

Bienvenido al repositorio oficial de PySI, una solución tecnológica diseñada para la optimización de procesos y la gestión eficiente de información. Este proyecto representa un prototipo funcional interactivo desarrollado de manera colaborativa.

Caracteristicas del Proyecto

Gestión Eficiente: Procesamiento y almacenamiento estructurado de datos.

Interfaz Amigable: Diseñado para facilitar la interacción del usuario final.

Arquitectura Limpia: Estructurado bajo buenas prácticas de programación en Python, facilitando su escalabilidad.

Control de Versiones Riguroso: Desarrollo gestionado de manera incremental mediante Git y GitHub.

Tecnologias y Herramientas Utilizadas

Lenguaje Principal: Python 3.x

Entorno de Desarrollo: Visual Studio Code

Control de Versiones: Git & GitHub

Instalación y Ejecución

Para poner en marcha el prototipo funcional en tu máquina local, sigue estos sencillos pasos:

Clonar el repositorio:

git clone https://github.com/juanpapito-art/pysi-proyecto.git


Navegar al directorio del proyecto:

cd pysi-proyecto


Ejecutar la aplicación:

python src/main.py


Nota: Asegúrate de tener instalado Python 3.x en tu sistema antes de la ejecución.
