# Registro de Eventos y Decisiones - Proyecto PySI

Este documento registra el proceso de desarrollo, reuniones y decisiones clave tomadas por el equipo para el diseño e implementación de PySI.

## Evento 1: Reunión de Alineación y Asignación de Roles
* **Fecha:** 28 de Mayo de 2026
* **Participantes:** Juan Manuel Londoño Rios y Cristian Duque Lenis
* **Detalles:** 
    * Se definieron los roles de trabajo para equilibrar la carga del proyecto de forma equitativa.
    * **Integrante 1 (Cristian):** Requisitos, Arquitectura, Pruebas y Documentación base.
    * **Integrante 2 Juan Manuel):** Prototipo, Administración de Git, Bitácora de Eventos e Implementación.
    * Se acordó utilizar Python como lenguaje principal debido a su versatilidad para el desarrollo de PySI.

## Evento 2: Definición de la Arquitectura y Estructura del Código
* **Fecha:** 31 de Mayo de 2026
* **Participantes:** Juan Manuel Londoño Rios y Cristian Duque Lenis
* **Detalles:**

    * Se estructuró el repositorio de Git con carpetas separadas para código (`src/`) y documentación (`docs/`).
    * Se resolvió trabajar con persistencia de datos local para el almacenamiento de datos de PySI.
